Create `frontend/src/app/core/services/auth.service.ts`:
- Injectable({ providedIn: 'root' })
- Private fields: inject(HttpClient), inject(Router)
- `isAuthenticated = signal<boolean>(false)` — Angular signal for reactive UI
- Constant `TOKEN_KEY = 'auth_token'`
- `async getToken(): Promise<string | null>` — calls `Preferences.get({ key: TOKEN_KEY })`, returns `.value`
- `async setToken(token: string): Promise<void>` — calls `Preferences.set({ key: TOKEN_KEY, value: token })`, sets `isAuthenticated` signal to true
- `async logout(): Promise<void>` — calls `Preferences.remove({ key: TOKEN_KEY })`, sets signal false, navigates to '/login'
- `login(email: string, password: string)` — returns `firstValueFrom(this.http.post<{data:{user:any,token:string}}>(\`\${environment.apiUrl}/auth/login\`, { email, password }))`
- `register(payload: FormData)` — returns `firstValueFrom(this.http.post<{data:{user:any,token:string}}>(\`\${environment.apiUrl}/auth/register\`, payload))`
- `getProfile()` — returns `firstValueFrom(this.http.get<{data:any}>(\`\${environment.apiUrl}/users/me\`))`
- `updateProfile(body: { name?: string; bio?: string; interests?: string[] })` — returns `firstValueFrom(this.http.patch<{data:any}>(\`\${environment.apiUrl}/users/me\`, body))`

Create `frontend/src/app/core/interceptors/auth.interceptor.ts`:
- Export `const authInterceptor: HttpInterceptorFn = (req, next) => { ... }`
- Inside: `const authService = inject(AuthService)`
- Return `from(authService.getToken()).pipe(switchMap((token) => { if (!token) return next(req); return next(req.clone({ setHeaders: { Authorization: \`Bearer \${token}\` } })); }))`
- CRITICAL: use `from()` to convert the Promise. Import `from` from 'rxjs', `switchMap` from 'rxjs/operators'.

Create `frontend/src/app/core/guards/auth.guard.ts`:
- Export `const authGuard: CanActivateFn = () => { ... }`
- Inside: `const authService = inject(AuthService); const router = inject(Router)`
- Return `from(authService.getToken()).pipe(map((token) => token ? true : router.createUrlTree(['/login'])))`
- Import `from` from 'rxjs', `map` from 'rxjs/operators', `CanActivateFn, Router` from '@angular/router'
- CRITICAL: return `router.createUrlTree(['/login'])` NOT `false`

Create `frontend/src/app/shared/components/avatar/avatar.component.ts`:

- Computed signals:
  - `initials = computed(() => this.name.split(' ').map(w => w[0]).slice(0, 2).join('').toUpperCase())`
  - `bgColor = computed(() => COLORS[hashName(this.name)])`

- Template: `*ngIf="!avatarUrl; else img"` div showing initials with `[style.background]="bgColor()"`, else `<ng-template #img><img [src]="avatarUrl" (error)="onError()" /></ng-template>`
