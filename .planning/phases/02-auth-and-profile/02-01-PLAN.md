Create `backend/src/middleware/auth.middleware.js`:
- Reads `Authorization` header; returns 401 with `{ data: null, error: 'UNAUTHORIZED', message: 'No token provided' }` if missing or not starting with 'Bearer '
- Extracts token, calls `jwt.verify(token, process.env.JWT_SECRET)`
- On success: attaches `req.user = decoded` (contains `{ id, email }`) and calls `next()`
- On jwt.verify error: returns 401 with `{ data: null, error: 'UNAUTHORIZED', message: 'Invalid or expired token' }`
- Exports: `module.exports = { verifyToken }`

Use `require('jsonwebtoken')` — do NOT use `require('bcryptjs')` here.
JWT_SECRET comes from `process.env.JWT_SECRET` — never hardcode.

Create `backend/src/services/auth.service.js`:
- Requires: `bcryptjs`, `jsonwebtoken`, `../config/db` (the pg Pool)
- `register({ name, email, password, avatarUrl })`: hashes password with `bcrypt.hash(password, 10)`, inserts into users table with `INSERT INTO users (name, email, password_hash, avatar_url) VALUES ($1, $2, $3, $4) RETURNING id, name, email, avatar_url, created_at`, returns row
- `login({ email, password })`: queries `SELECT * FROM users WHERE email = $1`, throws Error('Invalid credentials') if not found, calls `bcrypt.compare(password, user.password_hash)`, throws same error if mismatch, returns user row (caller strips password_hash)
- `signToken(user)`: returns `jwt.sign({ id: user.id, email: user.email }, process.env.JWT_SECRET, { expiresIn: '7d' })`
- Exports: `{ register, login, signToken }`

Create `backend/src/controllers/auth.controller.js`:
- `register(req, res)`: calls `authService.register({ name: req.body.name, email: req.body.email, password: req.body.password, avatarUrl: req.file?.path || null })`, calls `authService.signToken(user)`, responds 201 with `{ data: { user: { id, name, email, avatar_url }, token }, error: null, message: 'Registered' }`. Catch errors: if message includes 'duplicate' or 'unique' constraint respond 409; otherwise 500.
- `login(req, res)`: calls `authService.login({ email: req.body.email, password: req.body.password })`, deletes `user.password_hash` from the object, calls `authService.signToken(user)`, responds 200 with `{ data: { user, token }, error: null, message: 'Logged in' }`. Catch 'Invalid credentials' → 401; other errors → 500.
- Exports: `{ register, login }`

Create `backend/src/routes/auth.js`:
- Uses `express.Router()`
- Imports `{ uploadAvatar }` from `'../middleware/upload'`
- Imports `{ register, login }` from `'../controllers/auth.controller'`
- `POST /register`: `uploadAvatar.single('avatar')` middleware THEN `register` controller. Avatar is optional — if no file sent, `req.file` is undefined, which is handled by `req.file?.path || null` in the controller.
- `POST /login`: `login` controller (no upload middleware — login is JSON body)
- Exports: `module.exports = router`
